# E-Commerce-Analitico
Projeto de demonstração de um banco de dados relacional para um e-commerce. Inclui modelagem (DER), scripts de criação de tabelas, inserção de dados fictícios, views, procedures, triggers,e indices.

## Objetivo
Construir um projeto prático de SQL para demonstrar conhecimentos em:
- Modelagem relacional
- DDL e DML (CREATE, INSERT, UPDATE)
- Integridade referencial
- Procedures, triggers e views
